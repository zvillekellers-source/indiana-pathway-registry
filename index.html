🧭 Course Sherpa
AI-powered student diploma planning · Bring Your Own Key (BYOK) model
No student login required Sessionless · Email persistence
Step 1 — Enter API Key
Student or counselor enters their Anthropic API key. Key stored in browser memory only — never sent to IDOE servers.
Required disclosure: Your API key will be visible in browser network traffic. Set a spending limit on your Anthropic account before using this feature.
Step 2 — Upload Transcript (optional)
Upload a .json / .csv / .txt transcript file. Read by browser FileReader API — never sent to IDOE servers. Sent directly to Anthropic with user's key.
FERPA note: IDOE servers are not in the data chain. Browser → Anthropic directly.
Step 3 — AI Course Assistant
Student asks questions. Browser assembles: Sherpa API context bundle + transcript + conversation history → calls api.anthropic.com directly.
Quick questions: What courses do I still need? · Am I on track for Employment Honors? · What CTE pathway fits me?
